# 4. Series Operations In Pandas

## What is Series

Series is a single row or a single column in pandas.

```python
import pandas as pd
import numpy as np

d = ['iphone', 'motorolla', 'samsung']
s = pd.Series(d, name='phones')
s
```

Output:

```
0      iphone
1   motorolla
2     samsung
Name: phones, dtype: object
```
