# Microsoft Copilot for Fabric

Copilot in Fabric is an AI assistant that helps users work faster.

**Capabilities**

* Generate **SQL queries**
* Create **Power BI reports**
* Build **data pipelines**
* Explain datasets

**Example:**

```
Create a sales report by region
```

Copilot automatically creates visuals.
