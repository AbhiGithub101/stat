# 13. Dynamic Title

## What is a Dynamic Title in Power BI?

A **Dynamic Title** is a chart or visual title that **changes automatically** based on the user's selection, filters, slicers, or DAX measures.

Unlike a normal title (which always displays the same text), a dynamic title updates itself whenever the report data changes.

**In simple language, a Dynamic Title is an automatically changing heading that displays information based on the user's current selection in the report.**

### Why is a Dynamic Title Used?

Imagine a dashboard has data for all years.

If the user selects:

* 2023
* North Region
* Electronics

\-----> Instead of showing

Sales Report

\-----> the title automatically changes to

Sales Report - North Region (2023)

This helps users immediately understand what data they are looking at.



## Static Title vs Dynamic Title

| Static Title                  | Dynamic Title                |
| ----------------------------- | ---------------------------- |
| Fixed text                    | Automatically changes        |
| Same for everyone             | Changes according to filters |
| Manual update required        | Automatic update             |
| Less informative              | More informative             |
| Doesn't show filter selection | Shows current selection      |



### How Does a Dynamic Title Work?



<p align="center">User Selects Slicer<br>↓<br>Power BI reads the selected value<br>↓<br>DAX Measure generates text<br>↓<br>Visual Title changes automatically</p>







## How to Create a Dynamic Title

#### Step 1: Create a DAX Measure

Suppose you have a **Fiscal Year** column.

Dynamic Title =\
"Revenue Report - " &\
SELECTEDVALUE(DimDate\[Fiscal Year], "All Years")



#### Step 2: Select the Visual

Example:

* Line Chart
* Bar Chart
* Card
* Pie Chart



#### Step 3: Open Format Pane

<p align="center">Select Chart<br>↓<br>Format Visual<br>↓<br>General<br>↓<br>Title</p>



#### Step 4: Click the **fx** Button

Next to **Title Text**, click the **fx** (conditional formatting) button.



#### Step 5: Choose

<p align="center">Format by<br>↓<br>Field Value<br>↓<br>Select Measure<br>↓<br>Dynamic Title</p>

Click **OK**.

**Your chart title is now dynamic.**

## Common DAX Functions Used

| Function          | Purpose                                                      |
| ----------------- | ------------------------------------------------------------ |
| `SELECTEDVALUE()` | Returns the selected value from a slicer or filter.          |
| `VALUES()`        | Returns distinct values from a column.                       |
| `HASONEVALUE()`   | Checks whether exactly one value is selected.                |
| `CONCATENATEX()`  | Combines multiple selected values into a single text string. |
| `IF()`            | Displays different titles based on conditions.               |
| `SWITCH()`        | Changes titles based on multiple conditions.                 |



## Benefits of Dynamic Titles

* **Improves report readability** by showing the current context.
* **Clearly indicates active filters** such as Year, Region, or Product.
* **Reduces confusion** when users interact with slicers.
* **Makes dashboards more interactive** and user-friendly.
* **Eliminates manual title updates** when filters change.
* **Provides a professional look** for reports.
* **Useful in executive dashboards** where users frequently switch views.
* **Works seamlessly with bookmarks and slicers**.





## Best Practices

* Use **clear, descriptive titles** that reflect the selected context.
* Provide **default text** (such as "All Years" or "All Regions") when nothing is selected.
* Keep titles **short and readable**.
* Use **dynamic titles together with slicers** for better user experience.
* Test titles with **single selections, multiple selections, and no selection** to ensure they behave as expected.
