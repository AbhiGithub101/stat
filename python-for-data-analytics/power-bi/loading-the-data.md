# Loading the Data

**Loading data** means bringing data from a source (Excel, CSV, SQL Database, etc.) into Power BI so that you can analyze and visualize it.

#### Steps to Load Data

1.  **Open Power BI Desktop**

    * Start the Power BI application, click on **Blank Report** option



    <figure><img src="../../.gitbook/assets/image (70).png" alt=""><figcaption><p><strong>Home Page</strong></p></figcaption></figure>




2. **Click "Get Data"**
   * This option is available on the Home tab.
3. **Choose Data Source**
   * Select where your data is stored:
     * Excel
     * CSV
     * SQL Server
     * Web
     * Other sources
4. **Select the File or Database**
   * Browse and choose the file or connect to the database.
5. **Preview the Data**
   * Power BI shows a preview of the tables available.
6. **Load or Transform**
   * **Load**: Directly import the data into Power BI.
   * **Transform Data**: Clean or modify the data first using Power Query.
7. **Data is Loaded**
   * The tables appear in the Fields pane and are ready for reporting and visualization.
